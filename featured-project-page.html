Turn-Based Game Project

This project was for a client of mine. He wanted me to help him create this game for his university entrance exam. This game uses the Python module Pygame as its game engine. The game is turn-based, where the unit, whether enemy or yours, fills up its attack bar first can have a turn. It was a job request that I have taken on the freelancing website called Upwork.

I have worked on this project for three weeks at a rate of 4$ per hour. We arrange that the client will create the graphics of the game, the animation, and such. I will do the actual coding part, like adding the different features the client wants.
